## Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/library-nextjs.git
   cd library-nextjs
   ```
2. **Install dependencies**
   ```sh
   npm install
   # or
   yarn install
   ```
3. **Start the development server**
   ```sh
   npm run dev
   # or
   yarn dev
   ```
4. Open **http://localhost:3000/** in your browser.
